Keyboard Shortcuts :

On Linux :

Navigation :

  • pwd : print working directory
  • ls -F : list directories
  • cd dir : go down directory tree
  • cd .. : go up directory tree

Creating & Deleting :

  • mkdir : create a directory (folder)
  • mkdir -p test/a/b : create a nested directory
  • touch : create a file (add name.extention)
  • rm -r dir : remove a directory (dir)
  • rm file : remove (file)
  • rm -F file : force remove (file)

Moving & Paste :

  • mv a.txt dir : move a.text inside dir
  • cp a.text dir : copy a.text inside dir

Basic Git :

Remote Repository :

  • git clone + ssh of code repo
  • git push

Workflow :

  • git add --all
  • git commit --all

Checking-Status or Log-History :

  • git status
  • git log
